Vehicle Management System

Full-Stack Fleet Monitoring & Permit Renewal Engine

1. Working Principle

The system is designed as a full-stack web application that ensures data persistence and provides an AI-augmented user experience. It operates on four core pillars:

AI Smart Parsing Uses advanced regex patterns to interpret natural language commands, converting human sentences into structured database records automatically.
Permanent Storage Utilizes a Node.js Express server linked to a SQLite3 database file, ensuring that all data remains secure and persistent on the host machine.
Automated Status Engine Real-time logic calculates permit validity based on the system date, automatically highlighting vehicles expiring within 10 days.
Bulk Data Processing Integrated with SheetJS to allow seamless import and export of vehicle records between the database and professional Excel spreadsheets.

2. System Workflow

The following workflow defines the lifecycle of a vehicle record within the system:

Phase Action Technological Component
Data Entry Natural Language Input / Excel Upload AI Parser / SheetJS Library
Processing Validation & Status Calculation JavaScript Server Logic
Persistence SQL Insertion Express API -> SQLite3
